        Iris K-Nearest Neighbors Classification with Streamlit Frontend
Overview

This project demonstrates a simple Machine Learning classification app using the famous Iris Flower Dataset and the K-Nearest Neighbors (KNN) algorithm.
The application is built with a Streamlit frontend, allowing users to interactively input flower measurements and predict the iris species in real time.

The model classifies flowers into one of the following categories:

Iris Setosa
Iris Versicolor
Iris Virginica

Features

Train a KNN classifier on the Iris dataset
Interactive Streamlit web interface
Real-time prediction of iris species
Adjustable input sliders for flower measurements
Simple and beginner-friendly ML project

Technologies Used

Python
Scikit-learn
Pandas
NumPy
Streamlit
Matplotlib / Seaborn (optional for visualization)
